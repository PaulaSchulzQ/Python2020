@Media queries

CSS uses the @media rule to detect information about the device being used to acces the website. For example, it can be used to detectd the size of the device, its capatcta ans its orientation. For this course we will be using it to detect teh devid size, but keep in mind that the it is capable much more.

Desktop: Width greater than or equal to 1024px

Tablet: Width between 481px and 1023px

Phone: Width less than or equal to 480px
